# newbot-sourceio
S0urce.io mass bot

# To install

You need NodeJS (https://nodejs.org/en/download/)

Start `install.bat`


# Getting started

You can change various settings at the start of index.js

Start `start.bat`

type `help` for the help page

type `newbot ? x` to create x bots

type `hathack` to attack everyone
